L-System Rules Parser

This tool provides an interface to the lsystem-js library, allowing you to play with rules and combinations. To read up about L-Systems, http://en.wikipedia.org/wiki/L_Systems is a good place to start.

Some example configurations are provided. Click the name of the desired example below to load the configuration, and click on the "Parse" button

Examples:

  • Algae
  • Fibonacci
  • Koch Curve
  • Sierpinski Triangle
  • Dragon Curve
  • Fractal Plant

Configuration:

// the initial state to start the L-System, for e.g.: A
// the set of transform rules applied at each generation in JSON object format, for e.g.: {"A": "AB", "B": "A"}
// the number of generations to run the transformations, for e.g.: 3
